## Available Tools

- **Subnet Calculator** — calculate network address, broadcast, usable range, and hosts from any IP/CIDR
- **MAC Address Lookup** — identify device manufacturers by OUI prefix
- **Bandwidth Converter** — convert between Mbps, Gbps, MB/s, and other data rate units
- **Port Number Lookup** — search common TCP/UDP ports by number or service name
