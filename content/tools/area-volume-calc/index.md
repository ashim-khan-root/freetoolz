## FAQ

### What formulas does the calculator use?
Each shape uses its standard geometric formula. For example, circle area = πr², sphere volume = 4/3 πr³, cylinder volume = πr²h. The tool displays the formula alongside the result.

### Can I calculate area and volume at the same time?
The tool has separate modes for area and volume calculations. Switch between them using the mode toggle.

### What units are supported?
You can input dimensions in millimeters, centimeters, meters, inches, or feet. The result is shown in the squared or cubed version of your chosen unit.

### Is the formula shown for every calculation?
Yes. The calculator displays the relevant formula so you can verify the result and learn the math behind it.

## Tips

- Use the formula display as a learning tool — it helps you understand how the result was derived
- For irregular shapes, break them into combinations of basic shapes and add the results
- When calculating concrete or materials, always add 5-10% for waste and spillage
