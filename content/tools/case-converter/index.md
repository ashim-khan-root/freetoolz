## How to Use the Case Converter

1. **Type or paste** your text in the input box
2. **Click a case option** — UPPERCASE, lowercase, Title Case, Sentence case, or tOGGLE cASE
3. **Copy** the result instantly

All processing is done locally in your browser.

## Features

- **UPPERCASE** — convert all text to capital letters
- **lowercase** — convert all text to small letters
- **Title Case** — capitalize the first letter of each word
- **Sentence case** — capitalize only the first letter of each sentence
- **tOGGLE cASE** — reverse the case of each letter
- **One-click copy** — copy result to clipboard
