## How to Use the List Randomizer

1. **Enter items** — type each item on its own line in the text area (or paste from a spreadsheet)
2. **Choose action** — select Shuffle to randomize the entire list order, or Pick to select random items
3. **Set pick count** — if using Pick mode, choose how many items to select
4. **See results** — shuffled list or random picks displayed instantly
5. **Repeat if needed** — click again for a different result

Perfect for contest winners, random selections, team assignments, study order, and fairness in decision-making.

## Features

## FAQ

### Is the randomization truly random?
The tool uses the Fisher-Yates shuffle algorithm with cryptographically random number generation where available. For practical purposes, the results are statistically random and unbiased.

### Can the same item be picked twice?
When using Pick mode with multiple picks, each item can only be selected once. This prevents the same entry from winning a giveaway multiple times.

### What's the maximum number of items I can enter?
There's no hard limit, but performance is best with lists under 10,000 items. For very large lists, the shuffle might take a moment to process.

### Does the tool store my list?
No. All processing happens locally in your browser. When you refresh the page, your list is gone.
