## How to Use the Unit Converter

1. **Select a category** — choose from length, weight (mass), temperature, area, or volume. Each category has its own set of supported units.
2. **Enter a value** — type a number into any unit field within the selected category. All other units in that category update automatically to show the equivalent value.
3. **Read the results** — every supported unit for that category displays instantly. No need to select a "from" and "to" — just type any value and see all conversions at once.
4. **Copy any result** — click on any converted value to copy it to your clipboard for use elsewhere.
5. **Switch categories** — click a different category tab to start a new conversion without losing your previous results.

All calculations are done in your browser. No data is sent anywhere — your conversions are completely private.
