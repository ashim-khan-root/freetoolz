## How to Use the Cache Checker

1. **Enter a URL** — full or partial (protocol is optional)
2. **View cache links** — Google Cache, web cache, and text-only cache
3. **Click "Open in Google Cache"** — opens directly in a new tab
4. **Check other services** — Wayback Machine and other archive tools

## Features

- **Google Cache link** — direct link to cached version
- **Web cache link** — stripped version without site styling
- **Text-only cache** — plain text cached version
- **Other archive services** — Wayback Machine, CachedViews, and more
- **One-click open** — opens Google Cache in a new tab
