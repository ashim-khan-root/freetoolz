## FAQ

### How accurate is the online color picker?
Accuracy depends on your monitor calibration, but within the same device the picker returns precise HEX and RGB values that will be consistent across all applications.

### Can I enter a HEX code manually?
Yes. The tool has a manual input field where you can type a HEX code directly, and the picker will jump to that exact color.

### What's the difference between HSB and HSL?
HSB (Hue, Saturation, Brightness) is also called HSV. HSL uses Lightness instead of Brightness. HSB goes from black at minimum brightness to pure color at full brightness. HSL goes from black at minimum lightness through pure color at 50% to white at full lightness.

### Why does my screen show colors differently?
Every monitor renders colors slightly differently based on panel type (IPS, OLED, TN), color profile, and brightness settings. Colors from the picker are device-dependent unless calibrated.

### Can I use this for print design?
The picker shows sRGB values, which are for digital screens. For print work, use the values as a starting point and check against a printed swatch book.
