## How to Use the Password Strength Checker

1. **Type a password** — the analysis updates in real-time
2. **View strength** — visual meter from Weak to Very Strong
3. **Read feedback** — suggestions to improve your password

All analysis happens locally in your browser. Nothing is sent to any server or stored anywhere.

## Features

- **Real-time analysis** — updates as you type
- **Strength meter** — visual indicator with color coding
- **Character breakdown** — uppercase, lowercase, digits, symbols
- **Length analysis** — minimum length requirements
- **Entropy calculation** — bits of entropy estimate
- **Common patterns** — detects sequential, repeated, common passwords
- **Improvement tips** — actionable suggestions
