## How to Use the JSON Formatter

1. **Paste or type** your JSON in the input area
2. **Click Format** to beautify or **Minify** to compress
3. **Copy** the result with one click

Your data never leaves your browser. No server-side processing.

## Features

- **Format/Beautify** — pretty-print JSON with proper indentation
- **Minify** — compress JSON to a single line
- **Validate** — shows errors with line numbers
- **Error highlighting** — know exactly where the problem is
- **One-click copy** — copy formatted or minified output
