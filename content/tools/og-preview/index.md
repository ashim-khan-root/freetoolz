## How to Use the OG Preview Tool

1. **Enter page URL** — the domain will be extracted automatically
2. **Add OG title** — appears in bold in social shares (max 70 chars recommended)
3. **Write OG description** — appears below the title
4. **Set OG image URL** — see how the image renders in the card preview

## Features

- **Social card preview** — Facebook/LinkedIn/Slack style card
- **Live image rendering** — loads and displays your OG image
- **Title length checker** — warns if too long
- **Domain extraction** — automatic URL parsing
- **Error handling** — shows if image fails to load
