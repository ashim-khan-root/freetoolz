## How to Use the Regex Tester

1. **Enter your pattern** — without delimiters (e.g. `\d{3}-\d{4}`)
2. **Set flags** — g (global), i (case-insensitive), m (multiline)
3. **Add test string** — the text you want to search
4. **Set replacement** — optionally replace matches with a string
5. **View results** — match count, highlights, and replaced output

## Features

- **Real-time matching** — see matches highlighted as you type
- **All regex flags** — g, i, m, s, u, y
- **Match highlighting** — matched text is highlighted in the output
- **String replacement** — test regex replacement with backreferences
- **Error feedback** — invalid regex patterns show descriptive errors
- **Match count** — total number of matches found
