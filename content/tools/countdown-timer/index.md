## FAQ

### Does the countdown timer account for time zones?
The timer uses your device's local time. If you share a URL, recipients will see the countdown based on their own time zone, which may differ from yours. Factor this in for global audiences.

### Can I embed this on my website?
Yes — the countdown timer can be embedded using the provided embed code. It's a self-contained HTML widget that works on any site.

### What happens when the countdown reaches zero?
The timer stops at zero and displays a celebration message. It doesn't go negative or continue counting. Just reset or set a new date to start again.

### Does it work on mobile?
Yes — the timer is fully responsive and works on smartphones, tablets, and desktop browsers. The large display makes it especially readable on phones.

### Can I hide the seconds display?
The current version shows all units including seconds. For a less granular display (just days and hours), you can set the time to midnight and check back daily.
