## How to Use the Text Repeater

1. **Enter text** — type or paste the word, phrase, or character you want to repeat
2. **Set count** — choose how many times to repeat the text (up to 10,000)
3. **Choose separator** — pick from space, comma, newline, or enter a custom separator of your own
4. **Toggle count prefix** — optionally add line numbers to each repetition for numbered lists
5. **Generate** — the repeated output appears instantly with a live character count showing total length

Perfect for generating test data, creating separator lines, formatting repetitive content, and padding text.

## FAQ

### What is the maximum number of repetitions?
You can repeat text up to 10,000 times. Very large outputs (100,000+ characters) may cause browser performance limitations, so the tool includes a practical cap.

### Can I add a prefix or suffix to each repeat?
Use the "Count prefix" option to automatically number each line. For custom prefixes (like "Item: "), simply include them at the beginning of your text in the input field.

### Is my data stored anywhere?
No. Everything runs in your browser — no data is sent to any server. Your repeated text never leaves your computer.

### What separator should I use for different scenarios?
Use "newline" for vertical lists and ASCII separators, "comma" for CSV-style data, "space" for inline repeated words, and "custom" for specific formatting like HTML `<br>` tags or pipe `|` characters.

### Does the output preserve special characters?
Yes. Special characters (including symbols, emoji, and Unicode) are preserved in the output exactly as entered. The character count handles multi-byte characters correctly.
