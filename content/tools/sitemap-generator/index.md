## How to Use the Sitemap Generator

1. **Enter base URL** — your website URL
2. **List your page URLs** — one per line (path or full URL)
3. **Set options** — default priority, changefreq, lastmod date
4. **Copy the XML** — grab your complete sitemap

Perfect for SEO professionals launching or updating websites.

## Features

- **Per-URL customization** — set priority and changefreq per page
- **Auto URL resolution** — shorthand paths auto-expand with base URL
- **XML format** — generates valid sitemap XML
- **One-click copy** — grab the complete XML output
- **No data sent** — everything runs locally
