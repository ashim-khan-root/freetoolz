1. **Enter equipment power** — NVR power consumption in watts
2. **Add camera power** — number of cameras and wattage each
3. **Enter switch power** — PoE switch consumption in watts
4. **See results** — total load, UPS recommendations, and runtime estimates

Perfect for security system designers, facility managers, and integrators planning backup power for surveillance systems.
