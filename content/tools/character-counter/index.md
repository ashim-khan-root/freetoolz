## FAQ

### What's the difference between character count with and without spaces?
Character count with spaces includes every character including spaces. Without spaces removes all whitespace for a count of only visible characters. Both are useful depending on your requirements.

### Is this different from a word counter?
Yes. The character counter focuses on character-level statistics (total chars, chars without spaces, letter count) while also providing word and sentence counts for completeness.

### Is my data stored anywhere?
No. Everything runs in your browser — no data is sent to any server.

### What counts as a "letter" in letter count?
Letter count includes only alphabetic characters (A-Z, a-z). It excludes numbers, punctuation, spaces, and special characters. This is useful when you need a count of actual letters only.

### How is a "sentence" defined?
A sentence is any text ending with a period (.), exclamation mark (!), or question mark (?). The tool counts sentences by these delimiters, so abbreviations with periods may affect accuracy.

## Tips

- For SEO meta descriptions, keep them between 150-160 characters — this tool's real-time counter makes it easy
- When writing SMS campaigns, a single message is 160 characters; longer messages split into multiple segments
- Use the "without spaces" count when a platform counts only visible characters (common in some form validations)
