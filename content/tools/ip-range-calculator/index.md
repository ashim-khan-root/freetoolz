## How to Use the IP Range Calculator

1. **Enter CIDR notation** — e.g., `192.168.1.0/24` or `10.0.0.0/16`
2. **View network details** — network address, broadcast, first/last usable IP
3. **Check host count** — total and usable hosts
4. **See subnet mask** — dotted decimal and wildcard

## Features

- **Network & broadcast addresses** — calculated from CIDR
- **Usable IP range** — first to last usable host
- **Host count** — total and usable (subtracts network/broadcast for /31+)
- **Subnet mask & wildcard** — dotted decimal notation
- **Binary representation** — network address in binary
- **IP class detection** — Class A through E
