## FAQ

### What's the difference between HTML and XML sitemaps?
XML sitemaps are for search engines and contain technical metadata like last-modified dates. HTML sitemaps are for human visitors — they're actual web pages with clickable links organized for easy browsing.

### Will an HTML sitemap help my SEO?
Yes, indirectly. An HTML sitemap improves user experience and internal linking, which are positive signals for search engines. It also ensures deep pages get linked from a prominent page.

### Where should I put my HTML sitemap?
Most sites place it at `/sitemap` or `/site-map` and link to it from the footer. This makes it accessible from every page on your site.

### How many links should I include?
There's no hard limit, but aim for 100-200 links organized into groups. Keep it scannable — if you have thousands of pages, consider splitting by category.

## Tips

- Update your HTML sitemap whenever you add or remove major sections of your site
- Use descriptive link labels — "Web Design Services" is better than "Services Page"
- Keep groups to 10-15 links each for optimal scannability
