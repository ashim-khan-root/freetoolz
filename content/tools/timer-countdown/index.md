## How to Use the Countdown Timer

1. **Set the time** — use the hour, minute, and second controls to enter your desired countdown duration. You can click the up/down arrows or type directly into each field.
2. **Click Start** — the timer begins counting down immediately. The remaining time updates in real-time with a clear visual display.
3. **Pause or Resume** — need a break? Click Pause to stop the timer, then Resume to continue from where you left off.
4. **Reset** — click Reset to stop the timer and return to your original time setting without refreshing the page.
5. **Alarm sounds** — when the countdown reaches zero, an audio alarm plays and a flashing animation alerts you visually.

## FAQ

### How does the alarm work?
When the timer reaches zero, a series of beeps will sound. The beeps continue until you click Reset. Make sure your device volume is turned up if you need the audio alert.

### What is the maximum time I can set?
You can set up to 99 hours, 59 minutes, and 59 seconds — enough for overnight countdowns, long cooking processes like slow-roasting, or extended work sessions.

### Does the timer work in the background?
The timer continues counting down as long as the browser tab is open. Note that some browsers may throttle JavaScript in background tabs to save power, which could cause slight timing delays.

### Can I set multiple timers at once?
This timer supports one countdown at a time. For multiple simultaneous timers, run separate browser tabs.

### Does the alarm still sound if the tab is in the background?
Most browsers will still play the alarm sound when the timer ends, but some may delay audio playback for background tabs. For reliable alarms, keep the tab active.
