## How to Use the Subnet Calculator

1. **Enter an IP address** — e.g. 192.168.1.0
2. **Enter CIDR prefix** — e.g. 24 (or use netmask format like 255.255.255.0)
3. **View results instantly** — network address, broadcast, usable range, total hosts, and wildcard mask

Perfect for network engineers, system administrators, CCTV installers, and IT professionals planning IP networks for surveillance systems.

## Features

- **Instant calculation** — updates as you type
- **CIDR notation** — supports prefixes from /0 to /32
- **Complete details** — network address, broadcast, first/last usable IP, total hosts
- **Subnet mask** — both dotted decimal and CIDR
- **Wildcard mask** — useful for ACLs and routing protocols
