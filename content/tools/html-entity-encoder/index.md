## How to Use the HTML Entity Encoder/Decoder

1. **Choose mode** — Encode (text → entities) or Decode (entities → text)
2. **Type or paste text** — input appears in the output field in real-time
3. **Copy, swap, or clear** — buttons for common actions

## Features

- **Encode mode** — converts `& < > " '` and special characters to HTML entities
- **Decode mode** — converts entities like `&amp;` and `&#169;` back to characters
- **30+ character mappings** — common symbols, currency, arrows, quotes, math
- **Decimal entity support** — handles `&#123;` format for any character
- **Hex entity support** — handles `&#x3C;` format
- **Swap button** — exchange input and output
