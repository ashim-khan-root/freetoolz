## FAQ

### What's the difference between Mbps and MB/s?
Mbps = Megabits per second. MB/s = Megabytes per second. There are 8 bits in a byte, so 100 Mbps = 12.5 MB/s. Internet speeds are typically advertised in Mbps, while file downloads show MB/s.

### Why do I see different numbers in different tools?
Some tools use 1000-based (decimal) and others use 1024-based (binary) conversions. This tool uses the standard 1000-based conversion: 1 Gbps = 1000 Mbps, which matches how network equipment measures speeds.

### How do I convert my internet plan to actual download speed?
Divide by 8. A 500 Mbps plan gives you 62.5 MB/s maximum download speed. A 1 Gbps plan gives you 125 MB/s. This tool does this automatically.

### What's a good conversion to memorize?
1 Gbps = 1000 Mbps = 125 MB/s. This covers most modern network planning scenarios.

## Tips

- Double-check capitalization: MB vs Mb is a factor of 8 difference
