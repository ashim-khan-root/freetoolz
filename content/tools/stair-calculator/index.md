## How to Use the Stair Calculator

1. **Enter total rise** — floor-to-floor height
2. **Adjust target riser height** — optimal is 7 inches
3. **Set tread depth** — minimum 10 inches
4. **Get results** — number of steps, stringer length, total run

Perfect for builders, contractors, and DIY staircase construction. Follows standard 7-11 stair guidelines. All calculations in your browser.

## Features

- **Building code compliant** — follows IRC 7" max riser, 10" min tread
- **Stringer length** — exact cut length for stringers
- **Headroom check** — warns if clearance is low
- **Unit conversion** — inches or centimeters
- **Printable results** — clear breakdown of all measurements
